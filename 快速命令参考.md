# 🚀 日志查看器 - 快速命令参考

## 📋 常用命令

### 🎯 自动生成（推荐）

```bash
# 启动自动生成服务（只需运行一次）
./start_auto_viewer_service.sh

# 停止服务
./stop_auto_viewer_service.sh

# 查看服务日志
tail -f .auto_viewer_service.log
```

**说明**：服务会在后台运行，比赛结束后自动生成HTML文件。

---

### 🔧 手动生成

```bash
# 交互式手动生成（推荐）
./手动生成HTML.sh

# 直接生成所有比赛
python3 create_standalone_viewer.py
```

**说明**：立即为现有日志生成HTML文件，无需等待。

---

## 📂 文件位置

生成的HTML文件：
```
Config/Sim_Logs/[比赛时间]/[队伍名]/view_logs_standalone.html
```

**使用方法**：直接双击打开！

---

## 🎮 完整工作流

### 第一次使用

```bash
# 1. 启动自动生成服务
./start_auto_viewer_service.sh

# 2. 进行比赛（使用SimRobot）

# 3. 比赛结束后，文件管理器会自动打开

# 4. 双击 view_logs_standalone.html 查看日志
```

### 日常使用

```bash
# 服务已在后台运行，直接进行比赛即可
# 比赛结束后自动生成HTML
```

### 如果HTML显示0条记录

```bash
# 手动重新生成
./手动生成HTML.sh
```

---

## 🔍 故障排查

### 检查服务状态

```bash
# 查看服务是否运行
cat .auto_viewer_service.pid
ps -p $(cat .auto_viewer_service.pid)

# 查看最近的日志
tail -20 .auto_viewer_service.log
```

### 重启服务

```bash
./stop_auto_viewer_service.sh
./start_auto_viewer_service.sh
```
