# 日志文件时间戳功能

## 效果

日志文件名现在会自动包含创建时间（年月日时分秒）：

```
20260127_143025_fly11_fly11_Default_Default__Testing_2.log
└─ 2026年1月27日 14:30:25

20260127_150830_fly11_fly11_Default_Default__Invisibles_1stHalf_2.log
└─ 2026年1月27日 15:08:30
```

格式：`YYYYMMDD_HHMMSS_原文件名.log`

## 使用方法

1. 重新编译并部署到机器人
2. 新生成的日志会自动带时间戳
3. 直接从文件名看出每场比赛的时间，不用再猜

## 注意

- 旧日志不会重命名，只有新日志才有时间戳
- 确保机器人系统时间正确
