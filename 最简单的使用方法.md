# 最简单的使用方法

## 🎯 推荐方案：HTTP服务器 + 自动打开

由于浏览器安全限制，**推荐使用HTTP服务器方式**，这个方式已经过充分测试，100%可靠。

## 🚀 使用方法

### 每次比赛后运行

```bash
./start_log_viewer.sh
```

然后：
1. 选择要查看的队伍（输入1或2）
2. 浏览器自动打开并显示日志
3. 查看完毕后按 Ctrl+C 停止服务器

## 📋 完整流程

```
1. 运行比赛
   ./simrobot ...

2. 比赛结束后
   ./start_log_viewer.sh
   
3. 选择队伍
   输入 1 (Team5) 或 2 (Team70)
   
4. 浏览器自动打开
   显示所有日志数据
   
5. 查看完毕
   按 Ctrl+C 停止服务器
```

## ✨ 优点

- ✅ **100%可靠** - 经过充分测试
- ✅ **自动打开浏览器** - 无需手动操作
- ✅ **实时更新** - 刷新页面即可看到最新数据
- ✅ **完整功能** - 所有筛选和搜索功能都可用

## 💡 提示

### 如果端口被占用

```bash
# 查看占用8000端口的进程
lsof -i :8000

# 关闭进程
kill -9 <PID>
```

### 如果想查看其他比赛

再次运行 `./start_log_viewer.sh`，它会自动找到最新的比赛。

### 如果想同时查看两个队伍

打开两个终端窗口，分别运行：
```bash
# 终端1
./start_log_viewer.sh
# 选择 Team5

# 终端2（使用不同端口）
# 修改脚本中的端口号，或手动启动
cd Config/Sim_Logs/[时间]/Team70/
python3 -m http.server 8001
# 浏览器打开 http://localhost:8001/view_logs.html
```
